# Shape Drawing App

Controls:
Left Click Hold: Draw Shape
Left Click Release: Shape is drawn (now you can draw the next shape)
Num 1: Shape Rectangle
Num 2: Shape Circle
Num 3: Color White
Num 4: Color Blue
Num 5: Color Red
Num 6: Color Green
Num 7: Color Magenta
Num 8: Color Cyan
Num 9: Color Yellow
